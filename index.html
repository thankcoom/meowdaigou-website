📊 成本分攤方法

固定分攤

基礎

每個商品均攤相同的固定成本，適合商品價格相近的代購訂單。

計算範例：
情境：購買3個商品，總營運成本300元
計算：每個商品分攤 300÷3 = 100元
適用：商品價格相近，簡單快速

動態分攤

進階

根據商品價格比例分攤成本，高價商品承擔更多成本，更符合實際營運邏輯。

計算範例：
情境：商品A(1000元)、商品B(2000元)、商品C(3000元)
總值：6000元，營運成本300元
分攤：A: 50元 | B: 100元 | C: 150元
適用：商品價格差異大，分攤更公平
💡
選擇建議： 如果您的代購商品價格相近（如都是化妝品、都是服飾），建議使用「固定分攤」； 如果商品價格差異較大（如同時賣包包和小飾品），建議使用「動態分攤」。

💰 利潤設定方式

百分比模式

推薦

設定固定利潤率，自動根據成本計算利潤金額。適合想要穩定獲利率的代購業者。

設定範例：
利潤率：20%
商品A成本：1000元 → 利潤200元 → 報價1200元
商品B成本：500元 → 利潤100元 → 報價600元
優點：獲利率穩定，高價商品獲利更多

固定金額模式

簡單

每個商品加上固定利潤金額，不論成本高低。適合薄利多銷的經營策略。

設定範例：
固定利潤：150元
商品A成本：1000元 → 利潤150元 → 報價1150元
商品B成本：500元 → 利潤150元 → 報價650元
優點：計算簡單，低價商品獲利率高

混合模式

專業

結合百分比和固定金額，設定最低利潤保障。適合專業代購業者精細化管理。

設定範例：
設定：15% 或最少100元（取較高者）
商品A成本：1000元 → 15%=150元 > 100元 → 利潤150元
商品B成本：300元 → 15%=45元 < 100元 → 利潤100元
優點：保障最低獲利，高價商品按比例獲利

利潤設定建議

新手代購：建議15-20%，確保合理獲利
經驗代購：可用混合模式，最低利潤100-200元
高級商品：可降至10-15%，薄利多銷
特殊商品：可提高至25-30%，反映服務價值

🚚 運費計算設定

運費是代購成本的重要組成部分，MeowDaigou提供多種運費計算方式，協助您精確控制運輸成本。

🌍 國際運費

重量計費：適合服飾、小物件，按公斤計算
體積計費：適合大型商品，按立方公分計算
混合計費：取重量和體積費用較高者
範例：韓國寄台灣

商品：化妝品 500g

計算：500g × 每公斤運費 = 實際運費

分攤：如多件商品可選擇平均分攤或按重量比例分攤

🏪 本地配送

統一運費：所有商品相同配送費
免運門檻：滿額免運，提升客戶滿意度
地區差異：可設定不同地區不同運費
範例：台灣本地配送

設定：統一60元，滿1000元免運

訂單A：800元 + 60元運費 = 860元

訂單B：1200元 + 0元運費 = 1200元

📦
專業建議： 建議將包裝材料成本(10-20元)也計入運費中，確保所有相關成本都被涵蓋。 同時可考慮設定「代購服務費」來反映您的專業服務價值。
